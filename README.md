# 📊 사이버 범죄 데이터 분석 대시보드

## 📌 프로젝트 정보
- 기간 : 2025.04 ~ 2025.06  
- 참여 형태 : 5인 팀 프로젝트 (팀장)  
- 기여도 : 70%

## 📖 프로젝트 소개
사이버 범죄 데이터를 분석하여 유형별 통계와 연도별 추이를 시각적으로 제공하는 데이터 대시보드 서비스입니다.  
사용자가 범죄 유형과 기간을 선택하면 실시간으로 통계 결과와 시각화 차트를 확인할 수 있도록 구성하였습니다.

## 🧑‍💻 담당 역할
- 팀장 역할 수행 및 일정 관리  
- 데이터 전처리 및 통계 분석  
- Streamlit 기반 대시보드 구현  
- 시각화 차트 개발 (Matplotlib, Plotly)

## 📚 Lesson Learned
- 팀 프로젝트 리딩 및 협업 경험 강화  
- 대용량 데이터 전처리 및 통계 분석 실무 경험  
- 시각화 라이브러리 활용 역량 향상  
- 데이터 흐름 설계 및 사용자 중심 UI 구성 경험
